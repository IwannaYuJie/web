# 《雨姐的心动时刻》游戏开发文档 💕

## 📖 游戏简介

《雨姐的心动时刻》是一款以东北网红雨姐为原型的恋爱模拟游戏（Galgame）。玩家扮演一位来到东北的外国黑人小伙杰克，在雨姐的农家乐体验生活，通过不同的选择与雨姐建立感情，最终达成不同的结局。

## 🎮 游戏特色

### 核心玩法
1. **好感度系统**：通过选择影响雨姐对你的好感度（0-100）
2. **警觉度系统**：老蒯的警觉度会影响剧情走向（0-100）
3. **物品收集**：收集特定物品可以解锁隐藏结局
4. **时间推进**：游戏按天推进，共30-40天
5. **多结局系统**：5种不同结局，根据玩家选择决定

### 游戏结局
1. **真爱结局**：好感度≥90，警觉度≤30，第30天
2. **好友结局**：好感度60-89，警觉度≤50
3. **平淡结局**：好感度30-59
4. **被赶走结局**：警觉度≥80（坏结局）
5. **隐藏结局：三人行**：好感度≥95，警觉度≤20，收集所有物品，第40天

## 💻 技术实现

### 文件结构
```
src/
├── components/
│   ├── YujieGame.jsx          # 游戏主组件
│   └── YujieGame.css          # 游戏样式
├── data/
│   ├── yujieGameData.js       # 游戏基础数据
│   └── yujieGameEvents.js     # 游戏事件数据
└── pages/
    └── GameHub.jsx            # 游戏中心页面

public/
└── images/
    ├── 游戏图片说明.md        # 图片资源说明
    ├── character_*.png        # 角色立绘
    ├── scene_*.jpg            # 场景背景
    ├── cg_*.jpg               # CG图片
    ├── ending_*.jpg           # 结局CG
    └── item_*.png             # 物品图标
```

### 核心功能

#### 游戏状态管理
```javascript
{
  day: 1,                    // 当前天数
  affection: 0,              // 好感度
  laokuaiAlert: 0,           // 警觉度
  inventory: [],             // 物品栏
  currentEvent: 'event_id',  // 当前事件
  currentChapter: 0,         // 当前章节
  gameStarted: false,        // 游戏是否开始
  gameEnded: false,          // 游戏是否结束
  ending: null,              // 结局ID
  history: []                // 选择历史
}
```

#### 事件系统
每个事件包含：
- 场景设定
- 角色出场
- 对话内容
- 选择分支
- 效果影响

#### 结局判定
根据以下条件判定结局：
1. 好感度数值
2. 警觉度数值
3. 收集物品数量
4. 游戏天数

### 图片制作建议

#### 分辨率标准
- **16:9比例**：适用于场景、CG、结局（横向画面）
- **2:3比例**：适用于角色立绘（竖向画面）
- **1:1比例**：适用于物品图标（正方形）

#### 文件格式选择
- **JPG**：用于照片、场景、CG等色彩丰富的图片
- **PNG**：用于需要透明背景的图片（角色立绘、图标）

#### 文件大小控制
- 单张图片建议不超过2MB
- 总图片资源约60-100MB
- 可使用图片压缩工具优化

## 🎯 游戏攻略提示

### 真爱结局攻略
1. 尽量选择能增加好感度的选项
2. 避免让老蒯警觉度过高
3. 在关键时刻表白
4. 坚持到第30天

### 隐藏结局攻略
1. 收集所有6种物品
2. 好感度保持在95以上
3. 警觉度控制在20以下
4. 坚持到第40天
5. 在特定选项中选择特殊路线

### 避免坏结局
- 不要频繁选择增加警觉度的选项
- 适当安抚老蒯
- 送AD钙奶给老蒯
- 不要过于激进
