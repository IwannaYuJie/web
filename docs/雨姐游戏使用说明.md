# 《雨姐的心动时刻》使用说明 🎮💕

## 🚀 快速开始

### 1. 启动开发服务器

在项目根目录运行：
```bash
npm run dev
```

### 2. 访问游戏

打开浏览器，访问：
```
http://localhost:5173/secret-games
```

### 3. 开始游戏

1. 在游戏中心页面，找到"雨姐的心动时刻"卡片
2. 点击卡片进入游戏
3. 点击"开始游戏"按钮
4. 享受游戏！

## 🎯 游戏玩法

### 基本操作
- **点击对话框**：继续对话
- **选择选项**：点击你想要的选择
- **查看状态**：顶部状态栏显示好感度、警觉度等信息

### 游戏目标
通过与雨姐互动，提升好感度，同时控制老蒯的警觉度，最终达成你想要的结局。

### 状态说明
- **❤️ 好感度**：0-100，越高越好
- **⚠️ 警觉度**：0-100，老蒯的警觉度，过高会触发坏结局
- **🎒 物品**：收集的物品数量
- **📅 天数**：游戏进行的天数

### 选择提示
每个选择都会显示对状态的影响：
- **❤️+数字**：增加好感度
- **❤️-数字**：减少好感度
- **⚠️+数字**：增加警觉度
- **⚠️-数字**：减少警觉度

## 🏆 结局攻略

### 1. 真爱结局 💕
**条件**：
- 好感度 ≥ 90
- 警觉度 ≤ 30
- 第30天

**攻略提示**：
- 多选择增加好感度的选项
- 避免让老蒯起疑心
- 在关键时刻勇敢表白

### 2. 好友结局 🤝
**条件**：
- 好感度 60-89
- 警觉度 ≤ 50

**攻略提示**：
- 保持适度的亲密关系
- 不要过于激进

### 3. 平淡结局 😐
**条件**：
- 好感度 30-59

**攻略提示**：
- 这是最容易达成的结局
- 随意选择即可

### 4. 被赶走结局 😢
**条件**：
- 警觉度 ≥ 80

**攻略提示**：
- 这是坏结局，需要避免
- 不要频繁选择增加警觉度的选项
- 多和老蒯搞好关系

### 5. 隐藏结局：三人行 🎊
**条件**：
- 好感度 ≥ 95
- 警觉度 ≤ 20
- 收集所有6种物品
- 第40天

**攻略提示**：
- 这是最难达成的结局
- 需要完美平衡好感度和警觉度
- 收集所有物品：AD钙奶、猪肉、软枣、酸菜、柴火、野花
- 需要玩到第40天

## 🔧 常见问题

### Q: 游戏无法启动？
**A**: 确保已经运行 `npm install` 安装依赖，然后运行 `npm run dev`。

### Q: 图片显示不出来？
**A**: 这是正常的，游戏预留了图片位置。准备好图片后放到 `public/images/` 目录即可。

### Q: 如何重新开始游戏？
**A**: 点击右下角的"🔄 重新开始"按钮，或者在结局界面点击"重新开始"。

### Q: 如何查看我的游戏状态？
**A**: 顶部状态栏实时显示你的好感度、警觉度、物品数量和天数。

### Q: 选择会影响什么？
**A**: 每个选择都会影响好感度和/或警觉度，某些选择还会获得物品。

### Q: 如何达成隐藏结局？
**A**: 需要好感度≥95，警觉度≤20，收集所有6种物品，并坚持到第40天。

## 📝 开发说明

### 文件结构
```
src/
├── components/
│   ├── YujieGame.jsx          # 游戏主组件
│   └── YujieGame.css          # 游戏样式
├── data/
│   ├── yujieGameData.js       # 游戏基础数据
│   └── yujieGameEvents.js     # 游戏事件数据
└── pages/
    └── GameHub.jsx            # 游戏中心页面
```

### 修改剧情
编辑 `src/data/yujieGameEvents.js` 文件，添加或修改事件内容。

### 修改角色/场景
编辑 `src/data/yujieGameData.js` 文件，修改角色、场景、物品等数据。

### 修改样式
编辑 `src/components/YujieGame.css` 文件，自定义游戏界面样式。
